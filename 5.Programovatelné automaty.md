# Programovatelné automaty (PLC)

## 1. Programovatelné automaty (PLC)

Programovatelný automat (PLC – Programmable Logic Controller) je průmyslové řídicí zařízení určené pro řízení technologických procesů, strojů a výrobních linek. PLC je navrženo pro nepřetržitý provoz v průmyslovém prostředí a vyznačuje se vysokou spolehlivostí a odolností.

PLC:
- zpracovává vstupní signály
- provádí řídicí program
- ovládá výstupy podle logiky programu
- pracuje v reálném čase

Používá se především v průmyslové automatizaci.

---

## 2. Základní vlastnosti a rozdělení PLC

### 2.1 Základní vlastnosti PLC
- vysoká spolehlivost
- odolnost proti rušení
- dlouhodobý nepřetržitý provoz
- modulární nebo kompaktní konstrukce
- snadná rozšiřitelnost
- deterministické chování

---

### 2.2 Rozdělení PLC podle použití
- PLC pro jednoduché aplikace
  - řízení osvětlení
  - jednoduché stroje
- PLC pro středně složité aplikace
  - výrobní linky
  - dopravníky
- PLC pro složité aplikace
  - automatizované výrobní systémy
  - technologické celky

---

### 2.3 Rozdělení PLC podle velikosti
- malé PLC
  - omezený počet vstupů a výstupů
- střední PLC
  - možnost rozšíření
  - více komunikačních rozhraní
- velké PLC
  - stovky až tisíce I/O
  - vysoký výkon

---

### 2.4 Rozdělení PLC podle modulárnosti
- kompaktní PLC
  - CPU a I/O v jednom celku
  - menší rozšiřitelnost
- modulární PLC
  - samostatná CPU
  - samostatné I/O moduly
  - možnost rozšiřování podle potřeby

---

## 3. Popis základních částí PLC

PLC se skládá z několika základních částí, které zajišťují jeho funkci.

### Základní části PLC
- CPU (centrální procesorová jednotka)
  - vykonává program
  - zpracovává data
- paměť
  - paměť programu
  - paměť dat
- napájecí zdroj
  - zajišťuje napájení PLC
- vstupní moduly (DI, AI)
  - přijímají signály ze senzorů
- výstupní moduly (DO, AO)
  - ovládají akční členy
- komunikační rozhraní
  - Ethernet
  - PROFIBUS
  - PROFINET
  - RS-485

---

## 4. Základní diagnostika PLC

Diagnostika PLC slouží k odhalení chyb a poruch v systému.

### Základní diagnostické metody
- indikační LED diody na PLC
  - stav napájení
  - stav CPU
  - chybové stavy
- diagnostika v programovacím prostředí
  - stav vstupů a výstupů
  - chybová hlášení
- sledování proměnných online
- kontrola komunikace po síti

### Význam diagnostiky
- rychlé odhalení poruch
- zkrácení odstávek
- zvýšení bezpečnosti provozu

---

## 5. Programovací prostředí pro PLC

Programovací prostředí slouží k tvorbě, nahrávání a ladění programů pro PLC.

### Příklady programovacích prostředí
- TIA Portal (Siemens)
- Codesys
- Studio 5000 (Allen-Bradley)
- GX Works (Mitsubishi)

Programovací prostředí umožňuje:
- tvorbu programu
- simulaci
- online diagnostiku
- ladění programu

---

## 6. Programovací jazyky pro PLC

Programovací jazyky PLC jsou definovány normou IEC 61131-3.

### Základní programovací jazyky
- Ladder Diagram (LD)
  - grafický jazyk
  - podobný elektrickým schématům
- Function Block Diagram (FBD)
  - blokové schéma funkcí
- Structured Text (ST)
  - textový jazyk podobný Pascalu
- Instruction List (IL)
  - nízkoúrovňový jazyk (dnes již zastaralý)
- Sequential Function Chart (SFC)
  - řízení kroků a stavů

---

## 7. Závěrečné shrnutí

Programovatelné automaty jsou základním prvkem průmyslové automatizace. Vyznačují se vysokou spolehlivostí, odolností a možností nepřetržitého provozu. PLC lze rozdělit podle použití, velikosti a modulárnosti. Správná diagnostika a vhodné programovací prostředí a jazyky jsou klíčové pro bezpečný a efektivní provoz automatizovaných systémů.
