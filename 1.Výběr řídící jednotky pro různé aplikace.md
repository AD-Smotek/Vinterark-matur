# Řídicí jednotky – typy, programovací jazyky a vývojová prostředí

Řídicí jednotka je zařízení, které přijímá vstupní signály, vyhodnocuje je
a na základě toho ovládá výstupy. Používá se v automatizaci, průmyslu,
dopravě, budovách a řízení technologií.

---

## Reléová logika

### Charakteristika
- řízení pomocí relé a stykačů
- neobsahuje software
- funkce je dána zapojením vodičů

### Programovací jazyk
- žádný (neprogramuje se)

### Programovací prostředí
- elektrická schémata
- výkresy zapojení

### Použití
- jednoduché stroje
- zapínání motorů
- osvětlení

---

## Mikrokontroléry (MCU)

### Charakteristika
- malý programovatelný čip
- obsahuje CPU, paměť a periférie
- nízká spotřeba a cena

### Programovací jazyky
- C
- C++
- Arduino (zjednodušené C++)
- Assembly
- MicroPython
- Python

### Programovací prostředí
- Arduino IDE
- PlatformIO
- MPLAB X (PIC)
- Microchip Studio (AVR)
- Thonny (MicroPython)
- Visual Studio Code

### Použití
- domácí automatizace
- měření
- robotika
- výukové projekty

---

## Raspberry Pi Pico

### Charakteristika
- mikrokontrolér RP2040
- vhodný pro výuku i prototypování

### Programovací jazyky
- MicroPython
- C / C++

### Programovací prostředí
- Thonny IDE
- Visual Studio Code + Pico SDK

---

## PLC (Programovatelné logické automaty)

### Charakteristika
- průmyslové řídicí jednotky
- vysoká spolehlivost a odolnost
- dlouhá životnost
- modulární konstrukce

### Programovací normy
- IEC 61131-3

### Programovací jazyky PLC
- LD (Ladder Diagram)
- FBD (Function Block Diagram)
- ST (Structured Text)
- SFC (Sequential Function Chart)
- IL (zastaralý)

## FOXTROT PLC (Teco)

- modulární systém

### Programovací prostředí
- Mosaic

### Programovací jazyky
- ST (Structured Text)
- FBD (Function Block Diagram)
- LD (Ladder Diagram)
- CFC

### Použití
- automatizace budov

- regulace vytápění, osvětlení, technologie

## Průmyslový počítač (IPC)

### Charakteristika
- průmyslová verze PC
- běží na operačním systému
- často spolupracuje s PLC

### Programovací jazyky
- C / C++
- Python
- Java
- C#
- JavaScript

### Programovací prostředí
- Visual Studio
- Visual Studio Code
- Eclipse
- IntelliJ IDEA

### Operační systémy
- Windows
- Linux

### Použití
- SCADA systémy
- HMI panely
- databáze
- vizualizace

---

## Shrnutí
- reléová logika → žádné programování
- mikrokontroléry → C / C++ / MicroPython
- Raspberry Pi Pico → MicroPython + Thonny
- PLC → IEC 61131-3 + prostředí výrobce
- FOXTROT → české PLC + Mosaic
- průmyslový PC → klasické programovací jazyky a OS

Správný výběr řídicí jednotky závisí na aplikaci,
prostředí, spolehlivosti a nárocích na programování.
